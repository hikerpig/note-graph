# Note Graph

Note Graph is a generic visualization lib designed to show the structure of the document space and the relations between each doc.

This is a demo app of Note Graph, mainly for developing. Find more detail feature introduction and docs on the [github repo](https://github.com/hikerpig/note-graph).
