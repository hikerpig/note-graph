# NoteGraphView

The view of the graph.

Wraps a d3 [force-graph](https://github.com/vasturiano/force-graph) instance inside.

## Example

```ts
const notes = [
  {
    id: '9bc04ffc-9f87-5a23-ae2d-c0435696ba41',
    title: 'browser-extension',
    linkTo: [],
    referencedBy: [ '7c8ba189-e781-557e-88ab-7312ea0ecfc8' ]
  },
  {
    id: '7c8ba189-e781-557e-88ab-7312ea0ecfc8',
    title: 'frontend-index',
    linkTo: [
      '9bc04ffc-9f87-5a23-ae2d-c0435696ba41',
      'dba2f9a2-3c49-5ab0-ad02-869d1165dee0'
    ],
    referencedBy: []
  },
  {
    id: 'dba2f9a2-3c49-5ab0-ad02-869d1165dee0',
    title: 'web-annotation',
    linkTo: [],
    referencedBy: [ '7c8ba189-e781-557e-88ab-7312ea0ecfc8' ]
  },
]
const dataModel = new NoteGraphModel(notes)

const view = new NoteGraphView({
  container: document.getElementById('graph-view-container'),
  dataModel,
})
view.initView()
```
